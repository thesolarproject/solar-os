# Playback queue

Solar uses a **single mixed queue** for music files, podcast episodes, and Reach streams.

## Now Playing

- The home **Now Playing** shortcut appears when something is queued or playing.
- Tap while playing → player. Tap when idle with a saved queue → queue editor.

## Queue editor

- Two-line rows: title + artist/album.
- Current track shows a play/pause glyph; you can’t move the now-playing slot.
- Queue order is restored after reboot when possible.

## Repeat and clear

- Repeat off + last track finished → queue may clear.
- Explicit clear from the editor or context menu removes persisted state.
